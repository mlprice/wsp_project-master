Simplicity in Navigation


The navigation of this website is done through a simple bar navigation, that is custom styled for this particular company. The nav features a drop down menu, indicated my the small arrow next to the section that contains it. Users can move throughout the site by clicking on each section or the link within the drop down menu of a section.

Visualization:

The component consists of a [div] and a styled [ul], containing linked list items ([li], and [a]). The component also features a nested [ul] with more [li] and [a] elements.
