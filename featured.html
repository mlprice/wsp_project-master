My Featured Component:


The component I chose as my featured web design component is the "about us" component. To view the post about this, click here!

Visualization:

The component consists of a [div] section under another [div]. The "about us" probably includes some form of [article] or [p] tag to display the text, with a [button] and a link [a] tag to invite the user to learn more about the company.



To add to the previous, this "about us" looks to utilize CSS's column feature, in which the developer can add columns to style a chunk of [p] paragragph. The font size looks to be very similar to the typical 12 pt. font, in a sans-serif font that is easy to read on computer or mobile displays.
