Comments on the (photo) gallery:


This component is a beautifully done photo gallery featured on the site's front page. This site is used as a photographers portfolio, making the display of the photos a top priority. This photo gallery allows the user to view the photo right there on the front page, as well as scrolling through alternate photos in a bar below the larger displayed picture.

Visualization:

The component is comprised of, most importantly, the [img] that make up the gallery. The gallery itself is probably nested in a [div], each [img] has an alternate thumbnail [img] displayed in a bar at the bottom that links [a] to the larger [img], or at least triggers a javascript event to display the larger [img] in the [div].
